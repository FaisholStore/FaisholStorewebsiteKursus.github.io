Materi Kursus | Faishol_Web

HTML atau Hypertext Markup Language adalah bahasa pemrograman untuk membuat halaman web. Dalam kursus kami, kami akan membahas dasar-dasar HTML hingga HTML5 dan cara menggunakannya untuk membuat website yang menarik.

CSS atau Cascading Style Sheets digunakan untuk mengatur tampilan visual dari sebuah halaman web. Kami akan mengajarkan dasar-dasar CSS, termasuk pemilihan elemen, styling teks, tata letak, dan animasi.

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat website interaktif. Kami akan mengajarkan dasar-dasar JavaScript, termasuk variabel, kondisional, perulangan, dan event handling.

Bootstrap adalah kerangka kerja front-end yang populer untuk membangun tampilan web yang responsif dan menarik. Kami akan mengajarkan dasar-dasar Bootstrap, termasuk penggunaan grid system, komponen UI, dan responsive design.

jQuery adalah library JavaScript yang digunakan untuk mempermudah penggunaan JavaScript pada website. Kami akan mengajarkan dasar-dasar jQuery, termasuk seleksi elemen, event handling, dan animasi.

Node.js adalah platform untuk menjalankan JavaScript pada sisi server. Kami akan mengajarkan dasar-dasar Node.js, termasuk HTTP server, routing, dan middleware.
