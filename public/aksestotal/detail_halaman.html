Kursus HTML

Belajar HTML dengan Lebih Mudah dan Kreatif

Tentang Kursus HTML

Kursus HTML kami memberikan pengetahuan dan keterampilan dasar dalam membuat halaman web yang menarik. Anda akan belajar tentang tag-tag HTML, struktur dasar halaman web, serta cara menyematkan gambar, video, dan tautan. Dengan kursus ini, Anda akan dapat membuat halaman web yang kreatif, interaktif, dan responsif.

  • Belajar dasar-dasar HTML dengan pendekatan praktis dan langsung.
  • Menyusun struktur halaman web yang rapi dan terstruktur.
  • Menggunakan tag-tag HTML dengan tepat untuk memformat konten.
  • Memasukkan gambar, video, dan tautan untuk membuat halaman web lebih menarik.
  • Membuat tata letak responsif agar halaman web terlihat baik di berbagai perangkat.
Daftar Sekarang

Kurikulum Kursus

Modul 1: Pengenalan HTML Memahami dasar-dasar HTML, tag-tag dasar, dan struktur dasar halaman web. Mengenal atribut HTML dan penggunaannya. Menyusun paragraf, heading, dan list dalam HTML.
Modul 2: Menyematkan Media Menyematkan gambar dalam halaman HTML. Menyematkan video dari sumber eksternal menggunakan tag video. Menambahkan tautan (link) dalam halaman HTML.
Modul 3: Membuat Halaman Web Responsif Menggunakan media query untuk mengatur tampilan halaman web pada berbagai perangkat. Membuat tata letak responsif menggunakan grid system. Menggunakan CSS Flexbox untuk tata letak responsif.
